¿Como crear tu encuesta?

¿Que es encuest Creator?

Es una herramienta que esta pensda para crear encuestas y test profecionales para todo tipo de empresas y usuarios que necesiten crear test de manera rapida; contamos con plantillas, o herramientas que permitan crear tu platilla de manera personalizada, esto con el fin de mejorar

¿Como se usa?

Se deberá estar debidamente registrado en la plataforma para poder accder a la mayoría de herramientas que disone la pagina, y para los usuarios anonimos, solamente estará accesible la forma mas basica de hacer preguntas, o crear las cuestiones de verdadero y falso. Para acceder a esta, de debe ir a la esquina superior derecha, en el boton Crear o en el inicio, en la tarjeta de crear encuesta

¿como saber que está correctamente creada la encuesta?

si eres un usuario anonimo, solo podrá acceder medienante el link para editarla o compartirla. Si es usuario Registrado, podrá acceder a las herramientas y administrciones en la parte de perfil luego si quiere editar, eliminar, o crear nuevas, padrás hacerlo desde ese mismo menú.
